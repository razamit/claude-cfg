# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, assume good intent. Personal attacks, sustained disruption, and off-topic noise are not welcome in issues, PRs, or discussions.

## Reporting

If you experience or witness behaviour that violates this code, please contact the maintainer at **razamit23@gmail.com**. Reports are handled in confidence.

## Scope

This applies to all project spaces — issues, pull requests, discussions, and any other forum where people interact as part of the project.
